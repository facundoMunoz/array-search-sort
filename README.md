## Implemented Algorithms

- Search
   - Brute Force
      - Linear Search
   - Decrease and Conquer
      - Binary Search
- Sort
   - Brute Force
      - Selection
      - Bubble Sort
   - Decrease and Conquer
      - Insertion
   - Divide and Conquer
      - Merge Sort
